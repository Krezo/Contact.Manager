## Run project ##

- `rename ./backend/.env.example to .env`

- `set database DB\_\* settings`

- `create database "contactss`

- `create database "contacts_test"s`

- `npm install`


## Dev ## 

- ` npm run dev`

## Test

- `npm run test`
